DOKUMENTACJA API

punkt dostępowy

punkt dostępowy znjaduje się pod adresem:
http://localhost:3000/api


uwierzytelnianie

klucz dostępu do API można wygenerować za pomocą JsonWebToken. klucz powinien być przekazywany za pomocą nagłówka HTTP x-auth-token.


requesty

GET ---> http://localhost:3000/api/ -----> pobieranie listy tasków
POST ---> http://localhost:3000/api/ -----> dodawanie tasków
PUT ---> http://localhost:3000/api/:id -----> aktualizacja tasków
DELETE ---> http://localhost:3000/api/:id -----> usuwanie tasków


kody błędów

200 ---> OK -----> zapytanie zostało wykonane poprawnie
400 ---> Bad Request -----> któryś z parametrów przyjął nieprawidlową wartość
401 ---> Unauthorized -----> brak autoryzacji lub uprawnień do danego zasobu
403 ---> Forbidden -----> brak uprawnień do danej akcji
404 ---> Not Found -----> zasób nie został odnaleziony
500 ---> Internal Server Error -----> serwer napotkał niespodziewane trudności które uniemożliwiły zrealizowanie żądania
